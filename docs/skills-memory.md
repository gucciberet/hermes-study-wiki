# 스킬과 메모리

Hermes가 반복 작업을 더 잘 처리하도록 돕는 두 가지 기능입니다.

## 스킬: 작업 방법을 저장하는 설명서

**스킬(skill)**은 특정 일을 잘 처리하기 위한 절차와 주의사항을 담은 문서입니다.

예를 들면:

- GitHub 저장소 만들기
- 배포하기
- PDF에서 글 추출하기
- 스마트 조명 제어하기

스킬은 “무엇을 기억할까?”보다 “이 작업을 어떤 순서로 할까?”에 가깝습니다.

```bash
hermes skills list
```

## 메모리: 나와 환경을 기억하는 기능

**메모리(memory)**는 사용자의 선호나 반복적으로 필요한 환경 정보를 다음 대화에서도 참고할 수 있게 저장합니다.

좋은 예:

- “응답은 짧은 목록 형태를 선호함”
- “프로젝트 파일은 특정 폴더에 저장함”

피해야 할 예:

- 비밀번호, API 키, 인증 코드
- 며칠 뒤 의미가 없어질 작업 진행 상황

## 차이 한 줄 정리

```text
스킬 = 일을 하는 방법
메모리 = 사용자·환경에 대한 오래 가는 정보
```
