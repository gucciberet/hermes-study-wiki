# 메시지 앱에서 사용하기

Hermes는 터미널뿐 아니라 Slack 같은 메시지 앱에서도 사용할 수 있습니다.

## 어떤 점이 편한가요?

컴퓨터 앞이 아니어도 휴대폰에서 요청을 보내고 결과를 받을 수 있습니다.

- 링크나 파일을 보내 분석 요청
- 예약된 요약 받기
- 실행 결과와 오류 알림 받기
- 팀 채널에서 작업 요청 공유

## 기본 흐름

```text
메시지 앱에서 요청
→ Hermes가 필요한 도구 사용
→ 결과를 같은 대화에 전달
```

메시지 플랫폼 연결은 환경에 따라 설정 방법이 다릅니다. 현재 연결 상태와 설정은 다음 명령으로 확인할 수 있습니다.

```bash
hermes gateway status
hermes gateway setup
```

자세한 연결 방법은 [공식 메시징 문서](https://hermes-agent.nousresearch.com/docs/user-guide/messaging/)를 참고하세요.

## 다음 단계

이제 실제로 한 가지 작은 일을 요청해 보세요. 예를 들어 “현재 폴더에 있는 파일을 설명해줘”처럼 구체적인 요청이 시작하기 좋습니다.
