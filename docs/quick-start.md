# 빠른 시작

설치가 끝났다면 터미널에서 Hermes를 실행합니다.

```bash
hermes
```

이제 평소 대화하듯 요청하면 됩니다.

```text
현재 폴더의 파일을 간단히 설명해줘
```

```text
이 프로젝트의 테스트를 실행하고, 실패하면 원인을 알려줘
```

## 한 번만 물어보기

대화 화면에 들어가지 않고 요청 하나만 실행할 수도 있습니다.

```bash
hermes chat -q "현재 폴더의 구조를 설명해줘"
```

## 기억해야 할 기본 원칙

1. **목적을 먼저 말하기** — “무엇을 만들고 싶은지”를 알려주세요.
2. **중요한 조건을 함께 말하기** — 예: “파일은 건드리지 말고 분석만 해줘”.
3. **결과를 확인하기** — Hermes가 작업했다면 파일·명령 실행 결과를 함께 확인하세요.
