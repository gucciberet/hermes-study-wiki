# Hermes란?

**Hermes는 대화만 하는 챗봇이 아니라, 도구를 사용해 실제 작업을 수행할 수 있는 AI 에이전트**입니다.

쉽게 말하면 이렇게 생각하면 됩니다.

```text
내가 요청한다 → Hermes가 필요한 도구를 고른다 → 결과를 확인해 알려준다
```

예를 들어 Hermes는 파일을 읽고 수정하거나, 웹을 검색하고, 명령어를 실행하고, 반복 작업을 예약할 수 있습니다.

## 무엇이 다른가요?

일반 채팅 AI가 답변을 주는 데 집중한다면, Hermes는 답변에 필요한 **행동**도 함께 할 수 있습니다.

- 파일을 만들고 수정하기
- 웹에서 최신 정보 찾기
- 프로그램 실행과 오류 확인
- 여러 작업을 나눠 처리하기
- Slack 같은 메시지 앱에서 작업 받기

## 이 위키의 약속

여기는 공식 문서 전체를 그대로 옮기는 곳이 아닙니다. 처음 사용하는 사람이 헷갈리기 쉬운 핵심을 **짧고 쉬운 한국어**로 설명합니다.

기술적 세부사항은 항상 [Hermes 공식 문서](https://hermes-agent.nousresearch.com/docs/)를 기준으로 확인하세요.
